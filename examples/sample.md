# 把周复盘改造成一篇小红书图文

先把 `md -> 图文` 这一步做稳，后面的 script 和 skill 才有一个稳定的落点。

- 不需要先碰平台登录
- 不需要先把整个工作流一次设计完
- 可以先验证分页、模板和导出协议

# 技术选型

优先选现代但克制的方案：

- 核心逻辑收敛到一个 `core`
- Web 只做可视化外壳
- CLI 负责自动化和批量调用
- 渲染走统一模板，而不是三端各写一套
